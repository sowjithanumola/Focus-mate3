FocusMate - Offline Password Login (Option B)

How to use:

1. Extract the ZIP and open index.html in your browser.

2. Go to Login -> Create Account: enter username and password to create an offline account.
3. Click Login to start a session. Note: you must login each browser session (closing tab clears session).
4. Go to Progress to add daily entries, including Mistakes and Remark. Click 'Generate Suggestion' for an automatic improvement suggestion.
5. Export CSV to save or backup your data.

Security note: Passwords are hashed with SHA-256 before storing in localStorage. This is an offline, browser-only system and is not intended for production authentication on the web.
